# Granting Permissions to a User Group of a Domain<a name="en-us_topic_0057845623"></a>

## Function Description<a name="section495175389414"></a>

This interface is used to grant permissions to a user group of a domain. A role is a set of permissions and represents a group of actions.

## URI<a name="section3019338085013"></a>

-   URI format

    PUT /v3/domains/\{domain\_id\}/groups/\{group\_id\}/roles/\{role\_id\}
