# Deleting Permissions of an Agency on a Project<a name="en-us_topic_0079467627"></a>

## Function Description<a name="s48f1857f7f6d48a3985f62aa814b421d"></a>

This interface is used to delete permissions of an agency on a project.

## URI<a name="sfe5ded2143184f9d9e1ae057fbeec16c"></a>

-   URI format

    DELETE /v3.0/OS-AGENCY/projects/\{project\_id\}/agencies/\{agency\_id\}/roles/\{role\_id\}
